# Audio Files Required

Place the following audio files in this directory:

## Required Files

### 1. `beat.mp3` - Background Music
- **Type:** Electronic Ambient / Dark Industrial
- **Duration:** 60 seconds (loopable preferred)
- **BPM:** 80-100
- **Notes:** No vocals, subtle and professional

**Suggested sources:**
- Epidemic Sound
- Artlist
- Envato Elements

### 2. `pop.mp3` - UI Sound Effect
- **Type:** Soft notification sound
- **Duration:** < 1 second
- **Usage:** Pain points appearance

### 3. `success.mp3` - Achievement Sound
- **Type:** Positive chime/ding
- **Duration:** 1-2 seconds
- **Usage:** Solution reveal, positive results
